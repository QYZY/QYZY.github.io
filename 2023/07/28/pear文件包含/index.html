pear文件包含

发表于 2023-07-28 分类于 CTF

利用条件

安装了pear扩展 php开启了register_argc_argv选项

docker中默认都开启

register_argc_argv

在命令行模式下获取传入的参数，在web服务的情况下获取url中?后面的内容，无论是否有=号 在web中传参有几点需要注意的： =和&会被当成字符串解析 * 每个参数之间需要使用+分割，不能使用空格 * 它的解析不影响正常的GET传参

pearcmd.php

一般在docker中的默认安装目录为/usr/local/lib/php,也有可能因为版本不同在/usr/share/php/pearcmd.php

pear/pcel是一个命令行工具。如果我们能够包含其中的文件，并控制输入的参数，就可以通过它执行命令。

config-create

创建配置文件，可以在不出网的情况下挂马。
第一个参数为写入的内容，但必须以/开头，第二个参数为写入文件的路径。

发送时需要抓包发送，不然会被url编码 然后包含/tmp/test.php就可以RCE了

install

需要靶机出网

会从vps上下载test1.php到/tmp/pear/download/目录，我们可以使用--installroot选项指定安装目录
